# 全站交互动效动画方案

## 一、项目背景

基于 Vue 3 + Tailwind CSS + TypeScript 的个人博客，需为 4 个页面（首页、简历、经历、关于）设计统一的交互动效体系，提升用户体验与品牌质感。

## 二、设计方向

- **风格**: 精致极简 + 微物理交互
- **动效强度**: 6/10（流畅但不喧宾夺主）
- **技术约束**: 零额外依赖，纯 CSS + Vue 3 原生

## 三、全局动效体系

### 3.1 页面过渡动画

```
进入: opacity 0 + translateY(30px) + scale(0.98) → 正常
离开: opacity 0 + translateY(-20px) + scale(0.98)
时长: 400ms / 250ms
缓动: cubic-bezier(0.16, 1, 0.3, 1)
```

**文件**: `src/App.vue`

### 3.2 滚动触发动画

使用 `IntersectionObserver` 实现元素进入视口时的渐入效果：

```
初始状态: opacity: 0, translateY: 20-30px
触发后:   opacity: 1, translateY: 0
时长:     600-800ms
缓动:     cubic-bezier(0.32, 0.72, 0, 1)
延迟:     支持 stagger 级联（index * 100ms）
```

**复用 Composable**: `src/composables/useScrollReveal.ts`

### 3.3 导航栏滚动效果

```
顶部:     透明背景 + 细边框
滚动后:   毛玻璃背景(blur-xl) + 阴影 + 边框加深
过渡:     300ms ease-out
```

**文件**: `src/components/TopNavBar.vue`

## 五、组件级动效规范

### 5.1 按钮微交互

```css
/* 默认 */
transition: all 0.2s ease;

/* Hover */
transform: scale(1.05);
box-shadow: 0 2px 8px rgba(0,0,0,0.08);

/* Active */
transform: scale(0.95);
```

### 5.2 卡片微交互

```css
/* 默认 */
transition: all 0.3s cubic-bezier(0.32, 0.72, 0, 1);

/* Hover */
transform: translateY(-2px);
border-color: var(--border-hover);
box-shadow: 0 4px 20px var(--shadow-color);
```

### 5.3 链接下划线动画

```css
/* 默认 */
width: 0;
transition: width 0.2s ease;

/* Hover */
width: 100%;
```

## 六、性能优化策略

1. **GPU 加速**: 所有动画使用 `transform` 和 `opacity`
2. **will-change**: 动画元素添加，结束后移除
3. **IntersectionObserver**: 懒触发，避免滚动监听
4. **passive 事件**: 滚动监听使用 `{ passive: true }`
5. **减少重排**: 避免动画 `width/height/top/left`
6. **prefers-reduced-motion**: 尊重用户偏好

## 七、无障碍支持

```css
@media (prefers-reduced-motion: reduce) {
  *, *::before, *::after {
    animation-duration: 0.01ms !important;
    transition-duration: 0.01ms !important;
  }
}
```
