# 关于页面优化方案

## 一、当前页面诊断

### 1.1 布局结构
- 单列垂直流式布局，版心 max-width: 800px
- 区块顺序：Hero卡片 → 关于博客 → 技术栈 → 爱好 → 联系方式 → 社交链接
- 各区块间距 20px，整体偏紧凑

### 1.2 现有交互效果
- 3D Tilt 头像（鼠标跟随倾斜）
- 打字机签名（两行顺序显示）
- Spotlight 标签光斑（鼠标跟随）
- 磁吸联系方式（鼠标靠近吸引）
- 粒子爆发（复制时）
- 呼吸状态点
- 天气/音乐图标动画
- 发光社交链接边框
- 滚动渐入动画

### 1.3 视觉问题
- Hero卡片与下方区块视觉断层
- 技术栈标签排列不齐（流式换行）
- 联系方式占用过多垂直空间
- 社交区与Footer之间空白过大
- 标签颜色过多导致视觉噪音
- 缺乏视觉焦点和层次感

---

## 二、优化方向（基于 high-end-visual-design + design-taste-frontend + stitch-design-taste）

### 2.1 氛围定位
**Soft Structuralism + Editorial Luxury 混合**
- 干净、温暖、有质感的个人空间
- 像一本精心排版的个人杂志
- 密度适中（Density 5），既有呼吸感又不空旷

### 2.2 色彩系统优化

| Token | 当前值 | 优化后 | 说明 |
|-------|--------|--------|------|
| Canvas | #FFFFFF | #FAFAF8 | 暖白画布，更柔和 |
| Card Surface | #FFFFFF | #FFFFFF | 保持纯白卡片 |
| Text Primary | #1A1A1A | #1C1917 | 暖黑，不刺眼 |
| Text Secondary | #6B7280 | #78716C | 暖灰，更协调 |
| Text Muted | #9CA3AF | #A8A29E | 暖灰，层级柔和 |
| Border | rgba(0,0,0,0.08) | rgba(28,25,23,0.06) | 暖色边框 |
| Accent | #4A3728 | #57534E | 暖石色，更现代 |
| Dark Accent | #C4A882 | #D6D3D1 | 浅色模式下的强调 |

**深色模式**
- 背景：#0C0A09（极深暖黑）
- 卡片：#1C1917
- 文字：#FAFAF8
- 强调：#A8A29E

### 2.3 排版优化

**字体层级**
| 层级 | 当前 | 优化后 |
|------|------|--------|
| 姓名 | 26px / 700 | 32px / 700 / tracking-tight |
| 身份 | 13px | 14px / 500 |
| 签名 | 15px italic | 16px / 400 / 正常体（去掉italic） |
| 区块标题 | 11px uppercase | 12px / 600 / uppercase / tracking-wider |
| 正文 | 14px | 15px / leading-relaxed |
| 标签 | 12px | 13px |
| 联系方式 | 13px | 14px |

### 2.4 布局重构

#### Hero 区域
- **Double-Bezel 保留**，但优化比例
- 外层：padding 4px → 2px，更精致
- 内层：padding 24px 20px → 32px 28px，更大气
- 头像：88px → 96px，更有存在感
- 状态条：改为更紧凑的 pill 形状，去掉外边框

#### 关于博客
- 去掉独立卡片，改为**纯文本段落**
- 与 Hero 之间增加 40px 间距
- 文字左对齐，max-width 65ch
- 首行缩进或增加引号装饰

#### 技术栈 + 爱好 合并
- 两个区块合并为**"技能与兴趣"**一个区块
- 技术栈用**网格布局**（4列或自适应）
- 爱好用更小的标签，放在技术栈下方
- 去掉彩色背景，改用**单色边框标签**
- 悬停时显示品牌色（Spotlight效果保留）

#### 联系方式重构
- 从垂直列表改为**2x2 网格卡片**
- 每个卡片：图标 + 标签 + 值
- 减少垂直占用
- 磁吸效果保留

#### 社交链接
- 从水平排列改为**图标-only 圆形按钮**
- 悬停显示平台名称 tooltip
- 发光效果保留但减弱

## 四、预期效果

### 优化前
- 色彩偏冷，对比度生硬
- 标签五颜六色，视觉噪音大
- 联系方式占用大量垂直空间
- 各区块视觉权重相近，缺乏层次

### 优化后
- 暖色调系统，更柔和舒适
- 标签统一风格，悬停显色
- 联系方式紧凑网格化
- Hero 更突出，层次分明
- 整体像一本精致的个人杂志

- **Apple Design**: 暖白背景、充足的留白、细腻的动效
- **Notion 个人页**: 简洁的文本、清晰的层级
