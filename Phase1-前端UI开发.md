# Phase 1：前端全部页面 UI 开发

> 目标：用假数据/静态数据把所有页面 UI 搭建出来，网站可完整浏览。

---

## 1.1 项目初始化

- 安装项目脚手架
- 配置全局样式（亮色/暗色主题支持）
- 搭建全局布局组件（导航栏 + 页脚）
- 配置移动端响应式断点

---

## 1.2 首页 `/`

- 身份打字机效果（主标题 + 轮播副标题）
- 一句话个人介绍
- 技术栈标签云（悬停放大动画，点击跳转项目集并预筛选）
- 数据概览卡片（项目数/GitHub Stars/代码提交/运行天数，数字滚动递增动画）
- GitHub 贡献热力图（悬停显示当日贡献数）
- 精选项目速览（3-4 张横向卡片）
- CTA 按钮（查看项目 / 了解更多）
- 动态背景（粒子或渐变）

---

## 1.3 项目集 `/projects`

- 项目卡片自适应网格（桌面3列/平板2列/手机1列）
- 每张卡片：封面图、标题、简述、技术栈小标签、状态角标
- 分类筛选栏（全部/全栈应用/开源项目/学习项目/Vibe Coding）
- 技术栈标签联动筛选（支持多选）
- 搜索框实时过滤
- 点击卡片弹窗展开（完整描述、截图轮播、功能列表、GitHub/演示链接）
- 返回顶部 FAB 按钮

---

## 1.4 随记 `/notes`（前端 UI）

- 日历视图（月份切换，有记录的日期标记圆点，点击展开当日随记）
- 时间线视图（倒序排列，无限滚动加载）
- 随记卡片（日期、天气图标、温度、心情 emoji、位置、正文、配图）
- 图片点击灯箱放大
- 视图切换按钮（日历 ←→ 时间线）
- 此时使用假数据填充

---

## 1.5 关于我 `/about`

- 头像 + 个人简介区
- 技能雷达图/进度条（进入视口时绘制动画）
- 技术栈分类展示（语言/框架/工具/云服务分组）
- 经历时间线（交替左右排列，滚动滑入动画）
- GitHub 年度统计图
- 社交图标链接行（GitHub/掘金/邮箱/Twitter）
- 联系表单 UI（姓名+邮箱+内容，前端校验，此时提交按钮仅做 UI 无后端）

---

## 1.6 全局收尾

- 404 页面
- 页面路由切换过渡动画
- 暗色/亮色主题切换（跟随系统 + 手动切换）

---

**Phase 1 完成标志**：4 个页面 + 404 页面均可完整浏览，所有交互动画到位，使用静态假数据。
